_Martim Moniz put his foot in the door._

Como as faíscas de uma fogueira, os nossos nove grandes heróis emergem com sapatos de granito e um novo apetite para o fungo azul chamado arte.

> Esta é a cantiga da Galeria Martim Moniz,\
> Um lugar tão efémero que é apenas um chamariz.
> 
> Apontem os vossos punhais, vós grandes heróis,\
> Directamente para os inúmeros sóis.
> 
> Não tem há nada para alcançar,\
> Prepara-te só para desfrutar!
> 
> Apenas expectativas reviravoltas,\
> E também visões soltas e borbulhentas.
> 
> Os tempos rápidos não esperam por ninguém,\
> E nem aqui, nem ali, concluem.
